## FAQ
- 如何获取歌单的id？

    - iphone/android
        
        1.点开你想分享的歌单，点击转发后复制链接
        
        2.在浏览器中打开

        3.在连接url里含有“id=”后面的数字截取

    - windows
        
        1.在浏览器中打开歌单主页面
        
        2.将list_id后字段截取

## 自己搭建服务器？
- 你需要python基础 并有一定的捣鼓能力 这在多文件下载时速度会有很大提升！


1. 配置好网易云api服务，文档[在此](https://binaryify.github.io/NeteaseCloudMusicApi/)

1. 将源代码下载下来

2. 更改main.py中external后面的字符串（需要一定基础）

3. 运行并输入list_id
